Audit log

The audit log query view requires the SQLite backend. The current configuration uses the JSON-lines file backend.

To enable querying, set audit.db_path in config.yaml:

audit:
  db_path: "/var/lib/pds-pro/audit.db"

The file backend continues to work; you just can't query it from this UI. Tail the JSON-lines file with jq instead.
